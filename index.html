1. Tujuan

Setelah mempelajari modul ini, peserta didik diharapkan dapat:

  • Memahami dasar pemrograman web menggunakan HTML.
  • Mengenal berbagai tag dan elemen dasar HTML.
  • Menyusun halaman web sederhana menggunakan HTML.
  • Menyelesaikan latihan pemrograman dasar HTML secara mandiri.

2. Dasar Teori

2.1 Pengertian HTML

HTML (HyperText Markup Language) adalah bahasa markup untuk membuat halaman web yang dapat ditampilkan oleh browser seperti Google Chrome atau Firefox. Browser akan menerjemahkan kode HTML menjadi tampilan visual halaman web.
