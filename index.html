1. Tujuan

  • Peserta didik mampu menampilkan informasi sederhana menggunakan HTML.

2. Dasar Teori

HTML (HyperText Markup Language) merupakan bahasa markup untuk menyusun halaman web yang dapat dibaca oleh browser. Browser seperti Google Chrome atau Firefox akan menafsirkan kode HTML menjadi tampilan visual halaman web.

Beberapa Elemen HTML yang Sering Digunakan:

Tag Fungsi
<h1> - <h6>Judul atau heading
<p>Paragraf teks
<a>Hyperlink
<img>Menampilkan gambar
<hr>Garis horizontal
<br>Ganti baris
<b>, <i>, <u>Format teks (tebal, miring, garis bawah)
